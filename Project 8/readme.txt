Each query is wriiten into a seperate file.

To execute and store output in an xml file below is the command:
java -cp saxon.jar net.sf.saxon.Query -q:q1.xq -o:"o1.xml"

To exceute and view the results in the command line
java -cp saxon.jar net.sf.saxon.Query -q:q1.xq

please open the output file in Chrome browser for better readbility.
